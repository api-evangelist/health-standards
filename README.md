# Health Standards (health-standards)
Health Standards covers the collection of technical standards and specifications used in healthcare interoperability, including HL7 FHIR, HL7 v2, CDA, DICOM, IHE profiles, ICD codes, SNOMED CT, LOINC, RxNorm, US Core, and other health data exchange and coding standards. This index curates the canonical specifications, reference implementations, registries, and developer resources for building interoperable healthcare applications.
